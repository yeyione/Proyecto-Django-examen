Proyecto Django – Examen

Este proyecto fue desarrollado con Django como parte de un examen práctico.
Incluye un proyecto llamado "examen" y una aplicación llamada "parcial1" con dos vistas: home y about.


# Instalación y ejecución

1. Entrar a la carpeta del proyecto
cd examen

2. Crear entorno virtual
python -m venv venv

3. Activar entorno virtual en Windows
venv\Scripts\activate

4. Instalar Django
pip install django

5. Ejecutar el servidor
python manage.py runserver

# Rutas del proyecto

Home:
http://127.0.0.1:8000/

About:
http://127.0.0.1:8000/about/
